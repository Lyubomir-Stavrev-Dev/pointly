# Version 1.1 — App Store Connect checklist

Bundles four upgrades into one review. Build **1.1 (6)** produced by
`build_appstore.sh` → `build/appstore/Pointly.pkg`.

## 1. Upload the build
Drag `build/appstore/Pointly.pkg` into **Transporter** → Deliver. Wait ~15–30 min
for processing.

## 2. Create version 1.1 in App Store Connect
My Apps → Pointly Annotate → **(+) Version or Platform** → `1.1`.

## 3. Optimized metadata (ASO) — paste from `app-store-listing.md`
- **Name:** `Pointly Annotate: Screen Draw`
- **Subtitle:** `Laser pointer & spotlight live`
- **Keywords:** `highlight,marker,pen,whiteboard,presentation,teaching,demo,markup,blur,recording,arrow,notes`
- Keep the current Description (already has the EULA/subscription block from the
  3.1.2 fix).

## 4. App Preview video (big conversion win)
Media Manager → App Previews → upload
`marketing/video/out/appstore-preview-1920x1080.mp4`.
- App Store previews must be 15–30s ✓ (ours is 26s).
- Set the **poster frame** to the Spotlight moment.
- Reorder screenshots so the strongest (spotlight / draw) is first.

## 5. Select build 6, then Submit
- Build section → select **1.1 (6)**.
- "What's New in This Version" (paste):
  ```
  • Faster, smoother annotating
  • Quick upgrade options right from the menu bar
  • Polish and reliability fixes
  ```
- Submit for review.

## Notes
- Prices unchanged: IAPs stay $12.99/$39.99 base → €14.99/€46.99 in the EU. No IAP edits needed.
- Menu-bar Go Pro / Get Pro+ items are already in this build.
- No new permissions, no account, so review should be quick.
