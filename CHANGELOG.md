# Changelog

All notable changes to Pointly will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project structure with Swift Package Manager
- Overlay window system with screen recording permission handling
- Floating toolbar with draggable interface
- Core drawing tools: Pen, Highlighter, Eraser
- Shape tools: Rectangle, Ellipse, Arrow, Line
- Text tool placeholder
- Undo/redo functionality with state management
- Color picker and thickness slider
- Settings window with General, Appearance, and Drawing tabs
- Menu bar integration with system tray icon

### Technical
- SwiftUI + Combine architecture
- Canvas-based rendering system
- Immutable state management for undo/redo
- SF Symbols integration
- macOS 13.0+ compatibility
- Screen recording permission flow

## [0.1.0] - 2024-01-XX (Planned Alpha)

### Planned Features
- Basic pen and highlighter tools

- Minimal floating toolbar

- Menu bar presence
